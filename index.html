MenuBarFilter by eece

iOS-like dark menu bar for OSX

Installation

MenuBarFilter.zip
  1. Download and un-zip
  2. Drag to your Applications folder
  3. Then drag into your Login Items under System Preferences/Accounts
You may also want to un-check the "Translucent menu bar" option under System Preferences/Desktop

Source

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/eece/MenuBarFilter
